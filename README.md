# Luna 🌙

Luna is a personality-evolving desktop AI assistant designed to grow over time.

## Vision
Luna begins reactive and lightweight, then gradually evolves into a proactive, personality-aware assistant.

## Core Features (Planned)
- Adaptive personality system
- Memory persistence
- Behavior evolution logic
- Local-first architecture
- Modular core engine
